Hardware

Da ein Ziel des Projektes ist, die Hardwarekosten so gering wie möglich zu halten, gibt es keine hohen Anforderungen an die Hardware. Nutzen kann man hier einen Raspberry Pi und Raspberry Pi OS oder wahlweise ein v-Server mit einem Linux-System mit geringer Leistung.

Das System sollte mind. 50GB Speicher haben.

Betriebssystem

Prinzipell empfehlen wir ein Linux-System. Dadurch das unsere Anwendung aktuell nur als Containervirtualisierung läuft, ist es auch möglich Windows zu nutzen mit der zusatz Software Docker for Windows.

Als Distribution empfehlen wir Ubuntu oder CentOS. (Weitere Distributionen wurden nicht getestet)

Docker

Bitte nutzen Sie folgende Versionen:

  • Docker version 20.10.23
  • docker-compose version 1.29.2

Docker Desktop für Windows (Windows Linux Subsystem (WSL2)) wird ebenfalls unterstützt.
